# AX 컨설팅: 실무자를 위한 AI 기초와 응용

## 안내서 소개
이 안내서는 실무자가 **AI 기초를 이해하고 업무에 안전하게 적용**할 수 있도록 돕습니다.
복잡한 용어는 가능한 쉬운 말로 풀고, 꼭 필요한 용어는 **각 장의 용어 풀이**로 설명합니다.
실무 현장에서 즉시 활용할 수 있는 실천적 가이드를 제공합니다.

- **버전**: 2026-02-06
- **대상**: 실무자, 팀 리더, 실무 기획자
- **목표**: AI 리터러시 + 실무 적용 + 안전/윤리 기준 확립
- **먼저 읽기**: [프롤로그](prologue)

## 📖 읽는 방법
- **1~3장**은 기초 이해, **4~8장**은 실무 적용, **9~11장**은 확장/운영을 다룹니다.
- 각 장은 **개념 → 사례 → 한계/주의 → 용어 풀이 → 참고/출처** 흐름으로 구성되어 있습니다.
- 실습이나 제출물 없이 읽기만으로도 AI 기본 소양이 쌓이도록 설계했습니다.

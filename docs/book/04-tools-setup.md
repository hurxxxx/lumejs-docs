title: 04. 도구 선택과 셋업
---

## 학습 목표
- 업무 목적에 맞는 도구를 선택한다.
- 도입 전 체크리스트를 만든다.

## 핵심 개념
- **범용 챗봇**: 아이디어/요약/작성
- **문서/회의 도구**: 요약/정리
- **스프레드시트 AI**: 데이터 분석
- **자동화 도구**: 반복 업무 연결

## 예시
- 문서 요약은 범용 챗봇, 반복 업무는 자동화 도구와 결합

## 실습
1. 내 업무에 필요한 기능을 3가지 정의한다.
2. 보안/비용/내보내기 기준을 적는다.

## 체크리스트
- [ ] 데이터 반출 정책 확인
- [ ] 로그/접근 권한 확인
- [ ] 비용 구조 이해

## 요약
도구 선택은 "기능"보다 **보안/운영/확장성**이 중요하다.

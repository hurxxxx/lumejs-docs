## 학습 목표
- AS-IS/TO-BE로 업무를 설계한다.
- 자동화 우선순위를 정한다.

## 핵심 개념
- 트리거/조건/액션
- 예외 처리와 검증

## 예시
- 휴가 신청: 신청 -> 승인 -> 기록 -> 알림

## 실습
1. AS-IS 프로세스를 5단계로 정리
2. 자동화 후보를 3개 선정

## 체크리스트
- [ ] 예외 처리 흐름을 정의했다
- [ ] 검증/로그 단계가 있다

## 요약
자동화는 **작게 시작해서 반복 개선**한다.

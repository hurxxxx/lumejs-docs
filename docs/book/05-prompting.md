---
title: 05. 프롬프트 작성법
---

## 학습 목표
- 좋은 프롬프트의 구조를 이해한다.
- 업무에 맞는 템플릿을 만든다.

## 핵심 개념
- **역할**: 누구 관점으로 답할지
- **목표**: 기대 결과
- **입력/맥락**: 필요한 정보
- **제약**: 형식/길이/톤
- **검증**: 사실과 추정 분리

## 예시
```
역할: 영업 담당자
목표: 회의록 5줄 요약 + 액션 3개
입력: 회의록 원문
제약: 표 형식
검증: 사실/추정 분리
```

## 실습
1. 내 업무에 필요한 프롬프트 1개 작성
2. 제약 조건을 2개 이상 넣어본다

## 체크리스트
- [ ] 출력 형식이 명확하다
- [ ] 검증 기준이 포함됐다

## 요약
프롬프트는 **요구사항 명세서**처럼 작성한다.

## 학습 목표
- 데이터에서 질문을 만든다.
- 지표와 인사이트를 도출한다.

## 핵심 개념
- 지표(무엇을 본다)
- 비교(전주/전월/전년)
- 원인 가설(왜 변했는가)

## 예시
- 매출이 하락했다 -> 채널별/상품별로 분해

## 실습
1. 분석하고 싶은 지표 2개 정의
2. 변동 원인 가설 3개 작성

## 체크리스트
- [ ] 수치 기반 질문을 만들었다
- [ ] 추정과 사실을 구분했다

## 요약
데이터 활용의 핵심은 **좋은 질문**이다.

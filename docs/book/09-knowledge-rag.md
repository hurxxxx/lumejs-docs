title: 09. 지식 관리와 검색(RAG 이해)
---

## 학습 목표
- 지식 검색과 생성의 차이를 이해한다.
- RAG 흐름을 개념적으로 이해한다.

## 핵심 개념
- 지식베이스: 문서 저장소
- 검색: 필요한 정보를 찾는 과정
- RAG: 검색 결과를 기반으로 답변 생성

## 예시
- 사내 규정 질문 -> 관련 문서 검색 -> 요약 답변

## 실습
1. 자주 묻는 질문 5개 작성
2. 답변에 필요한 문서 목록 만들기

## 체크리스트
- [ ] 최신 문서 기준으로 답변한다
- [ ] 출처를 남길 방법이 있다

## 요약
RAG는 **정확한 근거 기반 답변**을 위한 구조다.

# AI Group Study (비전공자 → AX 컨설팅)

AI/코딩 경험이 없는 비전공자를 대상으로, **AI 기초부터 AX(AI Transformation) 컨설팅 실무 적용까지** 단계적으로 학습하는 6주 스터디입니다.

## 목적
- AI의 핵심 개념과 한계를 이해하고, 업무에 안전하게 적용할 수 있다.
- 프롬프트, 문서/데이터 처리, 간단한 자동화를 직접 설계·실습한다.
- 현업 문제를 정의하고 **컨설팅 관점의 문제분석/가설 수립**을 수행할 수 있다.
- **아키텍처·설계 사고**로 AX 관점의 해결안을 제안할 수 있다.

## 대상
- AI를 처음 접하는 비전공자
- 실무에 AI를 활용하고 싶은 직장인/소상공인/교육자
- 컨설팅 관점의 문제 해결을 배우고 싶은 사람

## 기대 효과
- AI 활용의 기본기(리터러시·윤리·한계)를 확보한다.
- 반복 업무를 AI/자동화로 개선하는 기초 실무력을 갖춘다.
- **문제정의/가설/KPI 기반**의 AX 제안서(미니 컨설팅)를 스스로 설계·발표할 수 있다.

## 운영 방식
- 기간: **6주 과정**
- 주기: **주 1회, 2–3시간**
- 구성: 짧은 강의 + 데모 + 실습 + 공유 + 과제
- 이론 미니강의로 **설계/컨설팅 프레임**을 먼저 잡고 실습에 연결
- 권장 인원: 8–20명 (토론/피드백 효율 기준)

## 커리큘럼 요약 (6주)
1. **AI 기초와 리터러시**
2. **실무 프롬프트 & 업무 자동화 기초**
3. **데이터/문서/스프레드시트 활용**
4. **노코드/자동화 워크플로우(프로세스/설계)**
5. **AX 컨설팅 기초(문제정의/이슈 트리/가설)**
6. **캡스톤: AX 컨설팅 제안(간단 아키텍처 스케치)**

## 운영 가이드
- **진행 방식**: 매 회차 실습 중심(50% 이상)으로 구성
- **과제/피드백**: 주간 과제 + 상호 피드백(짝/조별)
- **수료 기준**:
  - 출석 4회 이상(6주 기준)
  - 주간 과제 4회 이상 제출
  - 캡스톤 제안서 제출 및 발표

## 준비물
- 노트북, 안정적 인터넷
- Google 계정(문서/시트 활용)
- AI 도구 계정 1개 이상

## 안전/윤리 안내
- 개인/민감정보는 **가명 처리** 후 입력
- 회사/고객 데이터는 **사전 승인** 또는 **샘플 데이터**로 대체
- AI 결과는 반드시 **사람이 검토** 후 사용
